# Research Summary

This project explores the application of hybrid artificial intelligence models for network intrusion detection and anomaly analysis in real-time environments.

The system combines Long Short-Term Memory (LSTM) neural networks with Random Forest classifiers to improve anomaly detection performance on packet-based network traffic.

The research focused on:

* sequential traffic pattern analysis,
* anomaly detection in TCP/IP traffic flows,
* ensemble-based intrusion detection,
* reduction of false positives,
* real-time traffic monitoring and alerting.

The project was evaluated using the UNSW-NB15 dataset and achieved 99.20% classification accuracy during experimental evaluation.

The final system integrates:

* traffic parsing,
* feature extraction,
* machine learning and deep learning inference,
* alerting mechanisms,
* and a real-time visualization dashboard using Streamlit.

This repository represents the engineering implementation and experimental evaluation of the research system.
